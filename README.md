## Overview
A modern, fully responsive personal portfolio built to showcase my journey as a Backend Learner at HyperVerge Academy and a BCA student.  
The website highlights my projects, experience, certifications, and achievements through a clean, structured, and performance-focused design.  
It reflects my interest in scalable systems, API-driven applications, and real-world problem solving.

---

## Requirements
- A modern web browser (Chrome, Firefox, Edge, Safari)  
- Active internet connection for Google Fonts and EmailJS integration  
- No backend server required (fully client-side application)  

---

## Technologies Used
- **HTML5** for semantic structure and accessibility  
- **CSS3** with custom properties for dynamic light/dark theming  
- **Vanilla JavaScript** for UI logic, animations, and interactivity  
- **EmailJS** for contact form email delivery without backend setup  
- **Intersection Observer API** for scroll-based reveal animations  
- **LocalStorage API** for theme persistence  
- **Web APIs** for smooth scrolling, navigation highlighting, and UI state control  

---

## Key Features
- Fully responsive, modern UI  
- Light/Dark theme toggle with saved user preference  
- Smooth scroll navigation with active section highlight  
- Animated section reveals on scroll  
- Structured experience timeline  
- Project showcase with GitHub and live preview links  
- Certifications section with external verification  
- Functional contact form with toast notifications  
- Clean, minimal, and scalable architecture  

---

## Application Flow
- User lands on hero section with introduction and call-to-actions  
- Smooth navigation between About, Experience, Projects, Certifications, and Contact  
- Sections animate into view using Intersection Observer  
- Users can explore projects and external profiles  
- Contact form submits via EmailJS with instant feedback  
- Theme preference is stored for future visits  

---

## Purpose
- To present my developer profile in a professional and engaging format  
- To demonstrate frontend architecture, UI state handling, and API integration  
- To serve as a centralized digital presence for recruiters and collaborators

## Notes

- Built with scalability, performance, and clarity in mind
